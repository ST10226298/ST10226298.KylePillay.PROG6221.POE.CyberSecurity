
# CyberSecurity Awareness Chatbot

## Overview
This console-based chatbot helps users learn about basic cybersecurity concepts. It responds to common questions as showen down below and provides random security tips.

## Features
- Responds to user questions on cybersecurity.
- Provides random safety tips.
- Help command to guide user questions.
- Menu-based user interface.

## How to Run
1. Open the solution in Visual Studio.
2. Set `CyberSecurityAwareness` as the startup project.
3. Run the application.

## Example Questions
- What is phishing?
- How can I create a strong password?
- What is 2FA?
- What should I do if I get hacked?
